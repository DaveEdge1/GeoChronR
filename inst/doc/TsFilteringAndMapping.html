Now weâ€™ll filter it based on a range of criteria. For simple filtering, that include queries like > < or ==, you can use filterTs

## [1] "606 results after query: paleoData_iso2kPrimaryTimeseries == TRUE"

For more complex queries, however, itâ€™s often more efficient to pull vectors of the metadata that youâ€™d like to query and build and index

index <- which(between(lat,10,50) & between (lon,70,170) & var.name == "d18O")

OK, now weâ€™ve filtered down to what weâ€™d like to map. We can use mapTs() to map this.

You can change the mapping options by passing options to basemap() which creates the base map. Try ?basemap for options.
